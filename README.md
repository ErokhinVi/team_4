# Raiffeisen Bank — AI workshop, team repo

An AI workshop for the Raiffeisen bank board. The format is vibe-coding
with Claude Code (or Codex).

This repository is one team's territory. Four teams work in parallel —
each team in its own repository — and the simulator + leaderboard live in
the separate organiser orchestrator repo. All four teams solve the same
task. The bank is populated with simulated customers: every improvement
or breakage shows up in them — customers arrive or leave. A leaderboard
shows all four teams' scores.

The team is three service blocks (`retail`, `cib`, `backend`), one
participant per block. The team's three participants commit to the same
shared branch of this repo and an AI assistant helps them.

Roles and blocks — see `TEAM.md`, rules — `RULES.md`. For organisers,
deploy details and overall workshop layout — see the orchestrator repo
(`ORGANIZER.md`, `DEPLOY.md`).
